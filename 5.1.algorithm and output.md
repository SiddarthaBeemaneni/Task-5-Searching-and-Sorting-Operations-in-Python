5.1 Student Mark Search

Scenario: A class coach has marks of 50 students stored in a list. She wants to search for a particular student's mark to check if they passed.

Algorithm:

1. Store marks of 50 students in a list.

2. Input the mark to search.

3. Use linear search to check each element one by one.

4. If match found, print position and stop.

output:

enter the the mark to search:56

mark found at position 4
